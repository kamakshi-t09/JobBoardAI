# **🚀 JobBoardAI**

An AI-powered Job Portal built using Streamlit that helps job seekers find jobs, upload resumes, and receive ATS-based resume matching, while allowing recruiters to post and manage jobs.

✨ Features

- 👤 User Login & Signup
- 💼 Recruiter Job Posting
- 🔍 Job Search
- 📄 Resume Upload (PDF/DOCX)
- 🤖 AI ATS Resume Matching
- 🎯 Job Recommendations
- 📋 Application Tracking
- 👤 Profile Management

🛠️ Tech Stack

- Python
- Streamlit
- SQLite
- Pandas
- Scikit-learn
- NLTK
- PyPDF2
- python-docx

📂 Project Structure

```text
JobBoardAI/
│── app.py
│── requirements.txt
│── README.md
│── .gitignore
│── uploads/
│── jobboard.db
```

🚀 Installation

```bash
git clone https://github.com/kamakshi-t09/JobBoardAI.git
cd JobBoardAI

pip install -r requirements.txt

streamlit run app.py
```
